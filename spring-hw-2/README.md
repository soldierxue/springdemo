# springdemo

#spring-hw-2:

More complex User REST service, reconstruct the packages, demo the `@Controller、@RestController、@RequestMapping @ModelAttribute @PathVariable` usage.

The testing environment is :

```
1> Maven
2> Java 8
3> Tomcat Embed 
4> Spring boot 1.5.3
```

```
mvn clean package
mvn spring-boot:run
```
